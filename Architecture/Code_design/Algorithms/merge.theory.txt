GOAL ==>                          #Concatenating several sorted arrays into a single sorted array.
                                  #Keeps duplicates.

MERGE SORT ==>                    #Concatenate arrays then apply a merge sort
                                  #Time complexity: O(n * log n)
                                  #Space complexity: O(n)
                                  #Can be distributed

K-WAY MERGE ==>                   #Extract highest value among all k arrays, then iterate
                                  #Time complexity: O((n-k/2) * k)
                                  #Space complexity: O(n)
                                  #  - O(1) when using linked lists
                                  #Variants if high k:
                                  #  - store lists heads into a heap, making it O(n * log k)
                                  #  - divide and conquer, merging two arrays at a time
                                  #2-way merge is called "binary merge".
